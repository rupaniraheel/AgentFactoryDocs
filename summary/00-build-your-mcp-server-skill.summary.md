### AI Collaboration Keys
- Claude fetches MCP docs, asks clarifying questions, builds complete skill
- Each lesson tests and improves your skill
- "Reflect on Your Skill" sections at lesson end guide improvement

### Common Mistakes
- Skipping the skill creation step and jumping straight to code
- Assuming prior knowledge instead of building from official docs
- Not testing the skill before moving to lessons
